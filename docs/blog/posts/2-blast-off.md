# Blast Off!

Hey! Welcome back to the second blogpost on development on pre-alpha Apollo.

<!-- more -->

## Where are we?

Currently, I've created a base on the Apollo launcher. The GitHub repository containing the code can be found [here](https://github.com/tealingg/apollo-launcher "Take me there!"). This currently only launches Mario Kart Wii completely vanilla with no modifications.

One thing I was able to do was call OSReport after loading the game into memory. This is an encouraging sign as it means I'm able to call things such as a payload once it's created.

## Where are we going next?

I think I'd like to start visualising the road ahead for Apollo. Now that I've got a working launcher that runs and calls code I can start planning out exactly how I want the intricacies of Apollo's payload, launcher and patching mechanisms to function.
